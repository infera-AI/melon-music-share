# Melon Music Share

音乐分享H5页面项目

## 功能特性

- 🎵 音乐播放器
- 📱 响应式设计
- 🔗 应用跳转功能
- 🎨 现代化UI设计

## 部署状态

- 本地开发：`npm run dev`
- 本地预览：`npm run preview`
- GitHub Pages：https://xiguapopo67.github.io/melon-music-share

## 技术栈

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Howler.js
